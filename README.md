# 💱 Currency Converter

A modern and responsive currency converter built using React and Vite.
It fetches real-time exchange rates and allows users to convert between multiple currencies instantly.

---

## 🚀 Features

* 🔄 Real-time currency conversion
* 🌍 Supports multiple currencies
* ⚡ Fast performance using Vite
* 🎨 Clean and responsive UI
* 🔁 Swap currencies instantly

---

## 🛠️ Tech Stack

* React (Frontend)
* Vite (Build tool)
* JavaScript (ES6+)
* CSS (Glassmorphism UI)
* Frankfurter API (Exchange rates)

---
